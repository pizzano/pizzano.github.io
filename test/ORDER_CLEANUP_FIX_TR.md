# Sipariş ve admin temizlik düzeltmesi

- Valggrupper listesindeki hızlı `+ Legg til` butonları kaldırıldı.
- Manuel `Synkroniser` ve `Lagre nå` butonları admin üst köşesinden kaldırıldı.
- Müşteri sipariş bekleme/onay ekranında toplam tutar artık iki kez görünmez.
- Admin sipariş listesi daha güvenli hale getirildi; Firebase içinde bozuk/eksik kayıt varsa listeyi tamamen kırmaz.
- Siparişler varsayılan olarak özet kart şeklinde kalır, detaylar karta tıklayınca açılır.
