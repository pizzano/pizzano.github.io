# Logo boşken ne olur?

Logo URL boş bırakılırsa ana sayfada artık yazı, çizgi, KG/KØL benzeri özel logo gösterilmez.

Sistem şunu yapar:

1. Logo URL varsa: o logo görünür.
2. Logo URL boşsa ve Standard bilde URL varsa: o görsel üst alanda arka plan gibi kullanılır.
3. İkisi de boşsa: kategori kartlarında kullanılan sade default yemek görseli görünür.

Böylece sistemi başka restoranlara verirken KØL veya KG gibi özel yazılar görünmez.
