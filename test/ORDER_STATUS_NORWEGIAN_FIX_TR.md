# Sipariş status düzeltmesi

- Sipariş gönderildikten sonra sepet içinde artık toplam/form bilgileri görünmez.
- Müşteri sadece bestillingsstatus, fiş ve son siparişleri görür.
- Müşteri tarafındaki sipariş yazıları Norveççeye çevrildi.
- Canlı status popup yine açık kalır ve admin Godkjenn / Avvis yaptığında güncellenir.
