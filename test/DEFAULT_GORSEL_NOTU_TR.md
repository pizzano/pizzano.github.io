# Default görsel sistemi

Bu sürümde sistem başka restoranlar için daha hazır hale getirildi.

- `Logo URL` boşsa ana sayfada KØL'e özel sabit logo yerine genel restoran logosu görünür.
- `Standard bilde URL` alanına bir görsel URL yazarsan, resmi olmayan kategori/ürünlerde bu görsel kullanılır.
- `Standard bilde URL` de boşsa sistem CSS ile çizilmiş sade default yemek görseli gösterir.
- Bozuk resim URL siteyi bozmaz; otomatik default görsele düşer.
