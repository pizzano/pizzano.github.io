# Kom og hent + admin liste düzeltmesi

Bu sürümde:

- Müşteri Bestillingsstatus ekranında onaylanan sipariş için `Kom og hent om` alanı eklendi.
- Kalan süre dakika:saniye olarak kalın ve belirgin gösterilir.
- Süre dolunca müşteri tarafında `Maten er klar` gösterilir.
- Admin sipariş listesi sadeleştirildi:
  - Hele navn
  - Telefon nummer
  - Status etiketi
  - Varer / pris
  - Kalan süre
- Admin kapalı sipariş kartlarında tarih, açıklama ve tekrar eden gereksiz satırlar kaldırıldı.
- `fullName` alanı admin tarafında doğrudan kullanılır.
- CSS/JS cache versiyonları güncellendi.
