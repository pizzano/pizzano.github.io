# Popup admin ve canlı geri sayım düzeltmesi

Bu sürümde:

- Müşteri tarafındaki Bestillingsstatus ekranında onaylanan sipariş için `Kom og hent om` alanı daha belirgin yapıldı.
- Kalan süre dakika:saniye olarak büyük ve kalın görünür.
- Süre bitince müşteri tarafında `Maten er klar` görünür.
- Admin sipariş listesi sadeleştirildi:
  - Hele navn
  - Telefonnummer
  - Status
  - Varer / toplam fiyat
  - Canlı geri sayım
- Sipariş detayları artık liste içinde açılmak yerine tam ekran popup olarak açılır.
- Popup uzun siparişlerde kendi içinde scroll olur.
- Kabul / dakika / kanseller / yazdırma alanı altta sabit kalır.
- Sipariş onaylanınca popup kapanır ve sipariş listesine geri döner.
- Eski siparişlere tıklanınca aynı popup açılır, ama onaylama/dakika alanı gösterilmez.
- Üstte geri dön ikonu eklendi.
