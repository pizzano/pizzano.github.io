# menu-data.js olmadan çalışma notu

Bu sürümde `menu-data.js` zorunlu değildir.

- `admin.html` artık `menu-data.js` çağırmaz.
- Firebase tamamen boş olsa bile admin paneli açılır.
- İlk olarak **Legg til kategori** ile kategori ekleyebilirsin.
- Kategori ekleyince veri Firebase’e yeniden yazılır.
- Test menüsü otomatik geri basılmaz.

Eğer tüm veriyi kaybettiysen en güvenli yol: Admin > 💾 Yedek bölümünden eski JSON yedeğini geri yüklemek.
